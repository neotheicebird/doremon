Intro

  • This site keeps track of ideas discussed in Whatsapp group - GLStarters.
  • Doremon is a bot which listens to our converstions in whatsapp and whenever we
  • discuss ideas, it is going to write them down here in this website.
  • This project aims at creating a repository of the MAD ideas we come up with and
  • in time could guide any madster looking for some ideas.

    • Best practices to format messages

  • 1) Always make sure that the '#' and the 'tag' are NOT seperated by a `space`
  • Example: user-1 - we should make kids #sing and #dance this weekend to celebrate Valentine's day.

  • 2) When introducing an idea try to come up with an appropriate 'tag' and
  • when discussing on the same idea try to consistently use the same tag.
  • Example: user-2 - #sing is a nice idea. we should go for ARR songs!

    • Contribute

  • This project is in its infancy and could develop beautifully.
  • Please contribute your ideas in developing the project. Programmers and designers, you can just
  • start working on your ideas by creating a github account and forking Doremon
  • For more details contact Toptap.
